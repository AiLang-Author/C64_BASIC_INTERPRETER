check out ailang compiler, put these files in the ailang compiler folder add these librarys to the Librarys folder in the ailang compiler folder. 

from the ailang folder 

Ailang/ailang$     python3 main.py c64_basic_interpreter.ailang 

will create a exectuable with name c64_basic_interpreter_exec as the name 

run from terminal directory where you save file 

  ./c64_basic_interpreter_exec

60 GOLD = 0
70 KEYS = 0
80 ROOM = 1

120 GOSUB 400
130 IF HP <= 0 THEN GOTO 300
140 IF ROOM = 9 THEN GOTO 350
150 PRINT

170 PRINT "WEAPON: " + WEAPON$
180 PRINT
190 INPUT "CMD (N/S/E/W/Q)"; C$
200 IF C$ = "Q" THEN GOTO 300
205 IF C$ = "q" THEN GOTO 300
210 IF C$ = "N" THEN GOSUB 500
215 IF C$ = "n" THEN GOSUB 500
220 IF C$ = "S" THEN GOSUB 550

230 IF C$ = "E" THEN GOSUB 600
235 IF C$ = "e" THEN GOSUB 600
240 IF C$ = "W" THEN GOSUB 650
245 IF C$ = "w" THEN GOSUB 650
250 GOTO 120

310 PRINT "*** GAME OVER ***"

330 END

370 PRINT "GOLD: " + STR$(GOLD)
380 END
